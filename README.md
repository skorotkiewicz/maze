# Maze

A minimalist maze exploration game and level editor, crafted with precision and elegance.

## Features

- **Play**: Navigate through intricate levels with smooth, responsive controls.
- **Create**: Design your own mazes using the built-in intuitive editor.
- **Experience**: Enjoy a distraction-free interface with fluid animations and a refined aesthetic.

## Tech Stack

Built with modern web technologies for optimal performance and developer experience:

- **Core**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Build**: [Vite](https://vitejs.dev/)
- **Tooling**: [Biome](https://biomejs.dev/)

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/skorotkiewicz/maze.git
   cd maze
   ```

2. **Install dependencies**
   ```bash
   bun install
   ```

3. **Start the development server**
   ```bash
   bun dev
   ```

4. **Build for production**
   ```bash
   bun run build
   ```
